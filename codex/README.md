# OCI Deal Accelerator — Codex Skill Package

This directory packages the OCI Deal Accelerator as a portable AI skill.

## Quick Start

1. Point your LLM to `SKILL.md` as the system prompt
2. The `kb/` directory provides the knowledge base
3. Tools in `tools/` generate artifacts (.pptx, .drawio)

## Manifest

See `skill.json` for the skill definition including available tools.
